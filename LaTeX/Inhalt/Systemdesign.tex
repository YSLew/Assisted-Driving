\paragraph{Software}
Auf Softwareseite müssen sowohl das STM-Board als auch das FPGA-Board programmiert werden. Als Entwicklungsumgebungen werden jeweils Freewarelösungen verwendet. Das STM-Board wird mit der Entwicklungsumgebung \textit{CooCox CoIDE V1.7.7} in der Programmiersprache C programmiert. Diese Entwicklungsumgebung dient dem Programmieren und Debuggen von ARM Cortex MCU basierten Mikrocontrollerfamilien. Das FPGA-Board wird mit der Software \textit{Altera Quartus II Web Edition V11.0 SP1} in VHDL beschrieben. Die Software für den Logic Analyzer heißt \textit{Saleae Logic 1.1.15}.
